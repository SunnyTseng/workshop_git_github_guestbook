# Activities for git/github workshop! 

### Activity 1: Clone
- Copy the HTTP from the GitHub repo
- Open RStudio
- File → New Project → Version Control → Git
- Paste repository URL, choose a local path to store the repo
- You will see the repo (like a folder) in your local computer! 

### Activity 2: Guestbook
- Write down your github name (to be added as collaborator)
- Go to the local repo
- Open the README.md
- Write one sentence about yourself in that file
- Pull the latest changes from remote (to ensure the local is up to date)
- Commit and Push
